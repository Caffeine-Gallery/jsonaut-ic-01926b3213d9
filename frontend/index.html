JSON Path Accessor

Stored JSON:

Instructions

To access a specific JSON path, use dot notation for objects and bracket notation for arrays.

Default JSON structure:

{
  "name": "John Doe",
  "age": 30,
  "address": {
    "street": "123 Main St",
    "city": "Anytown"
  },
  "hobbies": ["reading", "swimming"]
}

Example paths:

  • name
  • age
  • address.street
  • address.city
  • hobbies.0
  • hobbies.1
